# 🎬 Netflix Content Analysis

## Overview
This project analyzes Netflix's content library to discover trends in movies, TV shows, genres, and release patterns. Using Python and Pandas, I explored what makes Netflix's catalog unique.

## Dataset
- **Source:** TidyTuesday (Netflix titles)
- **Records:** 7,787 titles
- **Columns:** 12 (title, type, genre, country, rating, release_year, etc.)

## Key Findings
- 📺 **Total Titles:** 7,787
  - 🎬 **Movies:** 5,377 (69%)
  - 📺 **TV Shows:** 2,410 (31%)
- 🎭 **Top 5 Genres:**
  1. International Movies (2,437)
  2. Dramas (2,106)
  3. Comedies (1,471)
  4. International TV Shows (1,199)
  5. Documentaries (786)
- 📅 **Top 5 Years with Most Releases:**
  1. 2018 (1,121 titles)
  2. 2017 (1,012 titles)
  3. 2019 (996 titles)
  4. 2016 (882 titles)
  5. 2020 (868 titles)
- ⭐ **Most Common Ratings:**
  - TV-MA (2,863)
  - TV-14 (1,931)
  - TV-PG (806)
  - R (665)
  - PG-13 (386)

## Key Insights
- 📈 **Peak Production Year:** 2018 with 1,121 titles
- 🌍 **International Content Dominates:** International Movies is the #1 genre
- 🔞 **Mature Content:** TV-MA is the most common rating (36.7% of titles)
- 📊 **Movies vs TV Shows:** Movies make up 69% of Netflix's catalog

## Technologies Used
- **Python 3** - Programming language
- **Pandas** - Data manipulation and analysis
- **Matplotlib** - Data visualization
- **Google Colab** - Development environment

The analysis includes charts for:
